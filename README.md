あみナビ (porokon)
概要
編み物初心者が挫折せずに作品を完成させるための支援アプリ「あみナビ」のソースコードです。
紙の編み図や英文パターンをカメラで撮影するだけで、自動でデータ化して編み進めるべき箇所をガイドします。

解決したい課題
編み図の読み取りにくさ: 複雑な記号や英文パターンの解読を自動化。

進行状況の紛失: 「今どこまで編んだか」をスマホ上で視覚的に管理。

担当範囲・役割
役割: フロントエンドおよびAPI連携の開発

具体的な実装:

Kotlin (Jetpack Compose) を用いた直感的なUI/UXの設計。

サーバーサイド（Go言語/Python）と連携するための APIクライアントの実装。

ViewModel による状態管理と、非同期処理（画像認識の結果取得など）のエラーハンドリング。

（2026年1月実装）「タイトルと製作者を覚える」機能: 作品の基本情報とクリエイター情報を保存し、後から簡単に呼び出せる管理機能の実装。

使用技術
Frontend: Kotlin (Jetpack Compose)

Backend: Go, Python (画像認識API)

Other: GitHub (チーム開発/バージョン管理), Retrofit (API通信)

技術的な工夫と解決した課題
API連携の安定化: 通信エラーや画像解析の待機時間が発生した際、ユーザーを不安にさせないためのローディング表示やリトライ処理をViewModelレベルで堅牢に実装しました。

UIの直感性: 編み物は手が塞がっていることが多いため、片手でも操作しやすいボタン配置や視認性の高いフォントサイズを意識しました。
